Arclix

Easy To Use React CLI

An Open Source CLI with Creation and Component Generation for React.

GET STARTEDGITHUB

Easy to Use

Learning or configuring anything new is not necessary. You can instantly initiate your work by creating a new project.

Rich Features

Out of the box support for Component Generation, TypeScript, CSS, SCSS/SASS, Test and more.

Trusty Dependencies

We use Create React App, TypeScript and SASS under the hood, which are reliable, well maintained and widely used.

Are You a React Developer?

If yes then you have arrived at one of the best CLI for component generation and creation.

When new versions of Arclix are released, you can upgrade using a single command:

npm install -g arclix@latest
