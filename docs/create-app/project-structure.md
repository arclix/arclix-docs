# Project Structure

After running the `create` command.

Inside the created directory, it will initialize the project and install the dependencies based on template:

### JavaScript Template

```
[PROJECT NAME]
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src
    ├── App.css
    ├── App.js
    ├── App.test.js
    ├── index.css
    ├── index.js
    ├── logo.svg
    └── serviceWorker.js
    └── setupTests.js
```

### TypeScript Template

```
[PROJECT NAME]
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src
    ├── App.css
    ├── App.ts
    ├── App.test.ts
    ├── index.css
    ├── index.ts
    ├── logo.svg
    └── serviceWorker.ts
    └── setupTests.ts
```
